# SubStreak Patch Notes

## 0.1.4

- The daily goal and streak are now separate overlay items you can arrange how you like — stack them, put them side by side, or justify left, center, or right.
- Grouped by default so they move together like a text block; ungroup to drag, scale, and rotate each one independently.
- New editor tools: one-click corner/center placement, grid and magnetic-alignment snapping, arrow-key and on-screen nudging, and exact position/scale/rotation entry.
- Add your own custom text lines to the overlay, with per-line size and color.
- The overlay editor canvas now gets much bigger when you maximize the window.

## 0.1.3

- Added a Dual PC / LAN overlay mode so OBS on another machine can use a private-network browser-source URL.
- Added an open-in-browser button next to the overlay URL copy button.
- Added a refresh button beside the app version so you can check for updates from inside SubStreak.
- The version indicator changes color when checking, when current, and when an update is available.

## 0.1.2

- Added a refresh button beside the app version so you can check for updates from inside SubStreak.
- The version indicator changes color when checking, when current, and when an update is available.

## 0.1.1

- Daily sub goals are now fully custom: type any whole-number goal instead of choosing from presets.

## 0.1.0

- First release: track a daily sub goal and your day-over-day streak.
- Log in to Twitch once; it stays connected.
- Counts subs, resubs, and gift subs automatically while you're live.
- Lives in the system tray — set it and forget it.
